# Fibonacci_Serisi_Akis_Semasi
Fibonacci serisinin hesaplanma mantığını canlı bir akış şeması üzerinden adım adım gösteren interaktif web uygulaması. Kullanıcının girdiği N değerine göre algoritma adımlarını (başlangıç, döngü, koşul kontrolü) görsel olarak vurgular, üretilen terimleri ve bunları klasik Fibonacci kare spirali üzerinde canlandırır.
